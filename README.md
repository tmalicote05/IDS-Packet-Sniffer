# IDS-Packet-Sniffer
A lightweight Intrusion Detection System (IDS) built in Python thay captures live network traffic and analyzes packets in real time using customizable threat detection rules.


# LEGAL NOTICE
This tool is intended for educational use and authorized network monitoring only. Any unauthorized packet sniffing may violate federal and local laws. Always obtain proper permission before capturing traffic on any network you do not own.
